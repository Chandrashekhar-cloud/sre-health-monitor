# Linux SRE Health Monitor

A Bash-based Linux monitoring tool that tracks CPU, RAM, Disk, Load Average, Network, and Processes.

## Features

* CPU, RAM, and Disk monitoring
* Slack webhook alerts
* HTML health reports
* Cron-based automation
* Structured logging
* Configurable thresholds

## Tech Stack

* Bash
* Linux
* Cron
* Slack Webhooks
* HTML

## Project Structure

* config.sh
* monitor.sh
* alert.sh
* report.sh
* install.sh
* logs/
* reports/

## Usage

```bash
chmod +x *.sh
./monitor.sh
./report.sh
```
